# Projektarbeit Digital Humanities 

## Modul: Einführung in die Digital Humanities 

### WS2025/26

#### Immanuel Kant - der geheime Verfasser der Briefe über die ästhetische Erziehung des Menschen?

##### Ein stilometrischer Vergleich ausgewählter philosophischer Schriften Schillers und Kants

Die vorliegende Arbeit untersucht aus syntaktischer Perspektive die Plausibilität der Annahme, dass das Werk Ueber die ästhetische Erziehung des Menschen in einer Reyhe von Briefen von Friedrich Schiller auch von dem zeitgleich wirkenden Philosophen Immanuel Kant verfasst worden sein könnte. Zur Untersuchung der Fragestellung wird eine quantitative stilometrische Analyse durchgeführt
